## Instalasi
1. Clone repositori ini:
    ```bash
    git clone https://github.com/kurapika12/TugasNella.git
    ```
2. Masuk ke direktori proyek:
    ```bash
    cd TugasNella
    ```
3. Instal dependensi yang diperlukan:
    ```bash
    npm install
    ```

## Penggunaan
Untuk menjalankan proyek, gunakan perintah berikut:
```bash
npm start
```
